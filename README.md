# ScenicSpotManagementSystem
基于WindowBuilder——景区信息管理系统  
    系统用户包括管理员和游客两类，管理员负责管理景区的景点维护；游客可以根据自己的需求对景区进行各种信息查询，及路线规划等。  
    主程序应用菜单选项包含功能：  
        -	管理员登录  
        -	景区景点分布图  
        -	景点的查找与排序  
        -	导游线路图  
        -	两个景点间的最短路径和最短距离  
        -	停车场车辆进出记录信息  
    管理员模块主要负责维护整个景区景点以及发布各项通知。管理员登录后可以进行：  
        -	景点的插入和删除  
        -	路的插入和删除  
        -	发布通知通告
